How To Add A Podcast

  1. Find the official RSS feed.
  2. Add a new item under `podcast_channels.selected_podcasts` in `config/sources.yaml`.
  3. Run `./run_daily.sh`.
  4. Push the updated repo so GitHub Pages republishes.

How To Add An X Source

  1. Add the account under `selected_x_accounts` in `config/sources.yaml`.
  2. Paste specific post URLs into `data/manual_sources.txt` until auto-ingestion is added.
  3. Run `./run_daily.sh` and push again.
